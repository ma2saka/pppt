# Pppt (Poor Point Presentation Tool)

Simple presentation tool on console.

## Installation

install it yourself as:

    $ gem install pppt

## Usage

Default presentation file is "presentation.yml"

    $ pppt init
    $ pppt show

You can specify for presentation file.

    $ pppt init my.yml
    $ pppt show my.yml
